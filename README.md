- [온라인 코드 리뷰 과정](https://github.com/woowacourse/woowacourse-docs/blob/master/maincourse/README.md)

## 기능 요구사항

* 참가자(Player)
    * 이름과 위치를 지닌다.
    * 주어진 방향으로 위치를 이동한다.

* 참가자들(Players)
    * 참가자들을 주어진 사다리를 기준으로 이동시킨다.

* 실행 결과(Result)
    * 실행 결과의 개수는 참가자들의 수와 동일해야 한다.

* 사람 이름 (Name)
    * 최대 5글자까지 부여할 수 있다.
    * 이름을 반환한다.
    * 이름의 길이는 최소 2글자여야 한다.

* 사람 이름들 (Names)
    * 중복을 허용하지 않는다.
    * 첫 번째 사람 이름을 반환한다.
    * 전체 이름 개수를 반환한다.
    * 특정 위치의 이름을 반환한다.
    * 이름 개수는 2개 이상이어야 한다.

* 위치(Position)
    * 현재 위치한 사다리의 순서.
    * 순서의 초기값은 이름을 입력한 순서이다.
    * 위치는 0이상이어야 한다.
    * 위치는 주어진 방향에 따라서 움직인다.

* 사다리 (Lines)
    * 라인을 모아둔 일급 컬렉션.
    * 사용자가 입력한 사다리 높이 만큼 라인을 생성한다.

* 사다리 높이 (Height)
    * 사다리 높이는 1 이상의 정수여야 한다.

* 라인 (Line)
    * 라인 폭은 사람 이름을 기준으로 한다.
    * 가로 라인은 겹치지 않도록 해야 한다.

* 입력
    * 사람 이름
    * 사다리 높이
    * 실행 결과

* 출력
    * 사다리 모양과 사람 이름을 함께 출력
    * 사다리 실행 결과를 출력한다.
        * 개인의 결과와 전체 결과를 선택할 수 있다.

* Parser
    * 사용자로부터 입력 받은 이름을 쉼표(,)를 기준으로 구분해서 반환한다.
    * 사용자로부터 입력 받은 사다리 높이가 제대로 된 숫자값인지 검증해서 반환한다.
